## 📝 Descrição do Projeto
Este programa foi desenvolvido em Python para a empresa de marketing **TudoWeb**. O objetivo é realizar uma pesquisa de opinião para medir o grau de satisfação dos clientes em relação ao atendimento prestado.

O projeto utiliza estruturas de repetição (`for`) e estruturas de decisão (`if/elif/else`) para coletar dados de múltiplos entrevistados e gerar um relatório estatístico.

## 🚀 Funcionalidades
- Coleta de dados: Nome, Idade e Opinião.
- Sistema de opções:
  1. **EXCELENTE**
  2. **BOM**
  3. **RUIM**
- Contagem automática de votos.
- Exibição de relatório final.

## 📊 Observações sobre o Relatório
Conforme solicitado nos requisitos do projeto, o relatório final exibe especificamente:
* A quantidade total de respostas **EXCELENTE**.
* A quantidade total de respostas **RUIM**.

**Nota:** Embora a opção **BOM** seja processada e validada pelo código (como pode ser visto na estrutura de decisão), ela não é detalhada no resumo final por uma definição de escopo da regra de negócio da TudoWeb.

## 🛠️ Como Testar
Para validar o funcionamento do programa, o laço de repetição foi configurado para **10 entrevistados**. 

1. Certifique-se de ter o Python instalado.
2. Execute o arquivo `.py` no seu terminal ou VS Code.
3. Insira os dados conforme solicitado no prompt.
4. Ao final da 10ª entrevista, o relatório será exibido automaticamente.
